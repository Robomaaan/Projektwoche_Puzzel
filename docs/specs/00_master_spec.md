# 00 Master Spec

PuzzleStudio ist eine lokale Admin-Dashboard-Webanwendung für registrierte Nutzer. Muss-Scope: Registrierung, Login, Logout, echte DB-Session, Bild-Upload, Bildverwaltung, Puzzle-Generierung, Spielansicht, Fortschritt speichern/fortsetzen, Accountverwaltung und QA.

Quelle: Pflichtenheft v2.0, Umsetzungsauftrag und DOT-Diagrammquellen.
